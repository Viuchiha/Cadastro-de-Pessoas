#  Sistema de Cadastro de Pessoas (Java Console)

Projeto desenvolvido para praticar Programação Orientada a Objetos (POO) e arquitetura em camadas (Model / Service / Repository) utilizando Java.

O sistema roda no terminal e permite gerenciar um cadastro simples de pessoas com operações de CRUD.

---

## Funcionalidades

Cadastrar pessoa  
Listar pessoas  
Buscar por nome  
Atualizar dados (nome, idade, email)  
Remover pessoa  
Validação de idade e email  
Menu interativo no terminal  

---

## Conceitos aplicados

Este projeto foi criado com foco em boas práticas de backend:

- Programação Orientada a Objetos
- Separação de responsabilidades (SRP)
- Arquitetura em camadas
- Collections (ArrayList)
- Encapsulamento
- Métodos getters/setters
- Validações de dados
- Uso de Scanner para entrada de dados
- Organização de pacotes
